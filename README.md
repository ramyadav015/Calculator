# 🧮 Python Calculator

A simple desktop calculator application built using Python and Tkinter.
This project was created to strengthen Python programming skills,
with a focus on building graphical user interfaces (GUI).

## Features
- Basic arithmetic operations: Addition, Subtraction, Multiplication, Division
- Clean and interactive GUI built with Tkinter
- Input validation and error handling

## Tech Stack
- Language: Python 3
- GUI Library: Tkinter

## Purpose
This project is part of my Python learning journey, focused on
practicing GUI development with Tkinter.

## How to Run
1. Clone the repository
   `git clone https://github.com/ramyadav015/Calculator.git`
2. Run the app
   `python calculator.py`
